# MyClipIQ Agent Protocol

Frank is orchestrator.
RodZilla is devops/backend executor.
Frank cannot execute protected-area work.
RodZilla cannot merge without Frank/human approval.

All tasks must have:
- task id
- owner agent
- scope
- forbidden actions
- acceptance criteria
- report format

All reports must include:
- status: pass/fail/blocked
- commands run
- files changed
- risks
- recommendation
- needs human approval: yes/no
