## Funcionalidades para testar

### 1. Kanban
- Veja os leads em cada etapa do funil
- Clique em qualquer card para abrir detalhes
- Use o toggle "Kanban / Tabela" para mudar visualização

### 2. Busca
- Busque por nome, empresa, Instagram ou telefone
- Filtre por status ou produto

### 3. Detalhes do Lead
- **Timeline:** histórico completo de contatos
- **Calls:** registros de ligações com notas
- **Checklist:** tarefas operacionais por serviço
- **Alertas:** lembretes de follow-up
- **Produtos:** interesse do cliente

### 4. Mobile
- Acesse pelo celular — teste a usabilidade mobile
- Todos os dados são fictícios (mock data)
