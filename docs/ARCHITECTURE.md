## Data Flow

1. **Intake:** Google Drive webhook вҶ’ MyClipIQ download вҶ’ R2 storage
2. **Editing:** Editor downloads from R2 вҶ’ edits вҶ’ uploads back
3. **Analysis:** FFmpeg extract audio вҶ’ Whisper transcription вҶ’ GPT analysis
4. **Review:** Editor picks AI suggestions вҶ’ Karine reviews вҶ’ Customer approves
5. **Posting:** TikTok API upload (inbox) вҶ’ Editor publishes from app
6. **Archive:** 30 days вҶ’ Move to Google Drive вҶ’ Delete from R2
